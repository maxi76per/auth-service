Лабораторная работа 1
Задание
Разработать защищенный микросервис аутентификации с реализацией современных механизмов безопасности.

1. Безопасная аутентификация
Реализовать регистрацию и вход с хешированием паролей (bcrypt/scrypt)
Добавить ограничение попыток входа (3 попытки/минуту)
Использовать JWT-токены с коротким временем жизни (15 мин)
2. Контроль доступа
Реализовать ролевую модель (user, moderator, admin)
Настроить проверку прав для защищенных маршрутов
Запретить параллельные сессии для одного пользователя
3. Защита данных
Обеспечить конфиденциальность паролей (хеш + соль)
Реализовать экранирование вывода пользовательских данных
Добавить логирование всех попыток входа
Этапы выполнения
1. Подготовка
Инициализировать проект Node.js
Установить зависимости:
npm install express bcryptjs jsonwebtoken express-rate-limit
